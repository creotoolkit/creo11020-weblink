Module "pfcScript"

Description

Emulates old style Web.Link methods and operations for the embedded browser.

Methods and objects in this module are deprecated and are provided for backward compatibility with old-style Web.Link pages. Equivalent object-oriented methods exist elsewhere in the library. PTC recommends that new development use the object-oriented classes and methods.
