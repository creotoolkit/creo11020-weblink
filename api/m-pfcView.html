Module "pfcView"

Description

Contains interfaces related to views in solid models.
Classes
classpfcView
This class specifies information about a view.
classpfcViewOId
This class specifies the identifier of a view. A view is identified by its name.
classpfcViewOwner
This is a base class for an object that has views associated with it (such as model).

Collections
classpfcViews
This data type is used to specify an array of views.
